# Object-Detection-Project
## Overview

This project detects Russian vehicle license plates using Haar Cascade Classifier and blurs the detected plate region using OpenCV.

## Requirements

* Python
* OpenCV
* NumPy
* Matplotlib

## Run the Project

1. Install required libraries
2. Load the image
3. Load the Haar Cascade XML file
4. Run the detection and blur function

## Output

The program detects the vehicle number plate and applies blur to hide the plate information.
